三姉妹

織田 奈那

こんばんわ♪

今日は私の姉妹について
お話したいと思います\( ˆoˆ )/

私は三姉妹の次女で
20歳の姉と
15歳の妹がいます

三姉妹とても仲が良いのですが
姉は友達とよく遊んでいたので
週末は家にいないことが多く
妹と過ごす時間が長かったです

小さい時は妹といると
すぐに喧嘩をしていたのですが

大きくなってからはそれがなくなり
妹の事が大大大好きになりました☆

妹は髪が茶色で顔が小さく
目が大きくてスタイルが良いので
とても羨ましいです( ˊᵕˋ )

でも最近食べ盛りなので
食べ過ぎないように気を付けてね☆

春からJKライフを満喫して下さい(^o^)

姉は妹や私とは真逆で
とても愛嬌があります❃︎
いつもいろんなところで
いろんな人と話しているのを見て
凄いなと思っていました( ˊᵕˋ )

私はすぐに姉の真似をしていたので
どこかで姉を尊敬していたのかなと
思います( ˊᵕˋ )

今でも沢山連絡を取り合っています☆

今は姉も1人で暮らし
私も東京に引っ越してしまったので
家族全員が集まる事が難しいですが
早くみんなで集まりたいです(●︎´∀︎`●︎)

いろいろ考えていたら
悲しくなってきたので
ここまでにしておきます( ･ᴗ･̥̥̥  )

それでは

おやすみすたーびーん
