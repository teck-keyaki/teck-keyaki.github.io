みなさん、こんにちは
福岡県出身、18歳の森田ひかるです 🌱










前回のブログの最後に書いていましたが、
ようやく紅葉を見に行く事が出来ました 。


毎年、見に行くので今年も行く事が出来て
満足です。









今度ライトアップされた紅葉も
見に行けたらいいなーと思います 🍁







もみじが赤、黄色、緑とグラデーションに
なっていてこの写真はお気に入りです。







東京でも田舎の方にくると安心しますね 〜


ファンの方が地元のここオススメですって
沢山教えて下さったので、いつか
行く事が出来るといいな〜 ( ¨̮ )













そして少し前ですが、まりな(松田里奈)と
お出かけしてきました 。


なんと私がずっと行ってみたいと
言っていた中華街です !




幸せな空間すぎて定期的に来ようって
話してました 〜 (笑)

小籠包とか角煮まんとか食べました〜 。
ずっと食べてみたかったものだったので
幸せでした 、、






遊園地にも行ったんです 。
オレンジと緑色のアトラクションで
やっぱり欅カラーの緑色に当たるよ〜って
話していたら見事2回ともオレンジでした (笑)






楽しかったです〜 !




















少し話しは変わりますが、
結構前のブログで漫画を買ったと
書いてたんですけど、ファンの方が
なんの漫画が気になります〜って
質問して下さっていたので紹介します〜 ( ¨̮ )






「EDENS ZERO」(エデンズゼロ) です!




真島ヒロさんの作品です!
私はフェアリーテイル大好き人間なのです。


もちろんその他作品も全て大好きです!




最新巻も発売日にゲットしてきました。
真島ヒロさんの単行本には
おまけページがあるのですが、それが
本当に大好きで1つの楽しみでもあります( ¨̮ )



オススメです 、、!










漫画は小学生〜中学生の頃
何百冊も集めていましたが、少年漫画が
ほとんどでした 〜







でもどのジャンルも好きです!



みなさんも何かオススメの漫画あれば
教えてくださいー ( ¨̮ )



















年内にたくさんブログかけたらいいなーと
思って最近かいています (*´`)/




目指せ10個以上ー!




基本ゆるいですが、暇な時に
ゆるく見て頂けたら嬉しいです 。












そして最後にお知らせです ✉️

今日「こちら有楽町星空放送局」が
オンエアされますー!
尾関さんと私が様々な質問に同時に答えて
シンクロするのか、という「こちほシンクロ」
をやっております ( ¨̮ )

23:30〜24:00オンエアです。ぜひ !







では今日はこのへんで。

最後まで読んで下さり、有難うございました。



おしまい 🌱
