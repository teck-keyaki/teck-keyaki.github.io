森田 ひかる







みなさん、こんにちは
福岡県出身、18歳の森田ひかるです 🌱









12月23日の「CDTV クリスマス音楽祭2019」
に出演させて頂きました!




「二人セゾン」を披露させて頂きました 。
大好きな楽曲なので、とても嬉しいです 。



今回は新しい衣装でした 。
すっごく可愛いですよね 。 ターンした時の
スカートの広がり方も綺麗で、ベルトも
ついてたり、おしゃれで可愛い衣装です !




いのうえ茶髪可愛いです ( ¨̮ )





あ、髪型はハーフアップにしました〜







見て下さった、皆さん有難うございました !

















そして少し遅くなりましたが、
12月16日に「ゆうがたパラダイス」に
井上、森田で出演させて頂きました〜



前回出させて頂いたのが、3月だったので、
やったことのない企画などもやったりして
とても楽しかったです 。



知恵の輪も解けました〜 ( ¨̮ )





金田さんがすごく面白くて、
あっという間な時間でした 。


そして吉本坂46さんの3rdシングルが
12月25日に発売されるそうです!
クリスマスですね!


「STRAY SHEEP」私とっても好きなんです 。

歌詞がとっても良いです 。
MVの世界観も !



MVはYouTubeにアップされているので
ぜひ見てみて下さい 。
金田さんがサムネイルになっています〜









ゆうパラさんまた来年もでれたら
嬉しいです ( ¨̮ )

メールも沢山、有難うございましたー!♩





















お話しは変わりますが、
今日はクリスマスイブですね  🎄



皆さんはパーティしますか??



イルミネーションも見にいけてない
です〜 。 さすがにイルミネーションは
1人で行く勇気はありません (笑)











あ、そういえば自分にクリスマスプレゼント
としてゲーム買いました 。



今はずっとポケモンしています ☀︎



最初のポケモンはメッソン !




私は、みずタイプ大好きなので
今までもずっとみずタイプのポケモンを
選んでました〜 。





特にエンペルトは大好きなんです 🐧















そして、最近、前髪がだんだん横髪に
なっていく現象が起こります (笑)


なのでぱっつんやめて久しぶりに
分けてみようかなと思います ( ¨̮ )



また気が向いたらしますね〜









あ、それとこの前、ファンレター
受け取ってきました 。


早く皆さんに会いたいな〜と
思いました ☺︎


それにクリスマスカード入ってて
びっくりしました!

他にも私が動物好きなので動物の便箋とか
シールとか沢山使って下さっていて、
とても癒されました 。


イラストとか本当に嬉しいです ( ¨̮ )


有難うございますー!大切にします !!
