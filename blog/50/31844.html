2019.12 7

FNS歌謡祭2019

森田 ひかる





みなさん、こんにちは
福岡県出身、18歳の森田ひかるです 🌱






先日、FNS歌謡祭2019 第一夜の
クリスマスメドレーに出演させて頂きました! 🎄









なんと大原櫻子さんとLiSAさんと
コラボさせて頂きました 。




夢のような空間でした 。

本当に有難うございました 。



一番最後のポーズの所で
欅ポーズしてきました ( ¨̮ )気づいた方いますか?









円卓にも座らせて頂きましたが、
とてもとても緊張しました 。
沢山のアーティストさんのパフォーマンスを
生で見る事ができ、凄く感動しました… 。

こんな幸せな事はないですね 。












そして今回は二期生の6人だけでの
出演というのが何より、緊張しました 。

普段どれだけ先輩方に助けて頂いているかを
実感しました 。




前日に先輩方とお仕事をしていたので、
その際に「明日頑張ってね!」「見守ってるよ〜」
と声を掛けて下さったり、連絡を下さったり
した事がとても励みになりました 。


本当にいつも有難うございます (*_*)








とはいえ、とても貴重な体験をさせて
頂くことができ、幸せでした 。




有難うございました !










12月11日(水)のFNS歌謡祭2019 第二夜、
欅坂46が出演させて頂きますので、
ぜひ見てください ( ¨̮ )
















そしてお知らせです〜 。


少し遅くなってしまいましたが、
欅坂46の2020年度公式カレンダーが発売決定
いたしました!



◎壁掛け大サイズ(B2縦)

→夏の全国アリーナツアー2019(東京ドーム公演)




◎壁掛け中サイズ(A4見開き)

→二期生自撮り




◎卓上サイズ

→3rd YEAR ANNIVERSARY LIVE






の3種類となっています。
ライブ写真が使用されているので、
とてもかっこいいカレンダーに
なっていると思います 。







そして今回、二期生だけの自撮りカレンダー
もあります !


月ごとに衣装を変えたり、行事にそった
アイテムを持って撮影したり、とても
楽しかったです 。








夏鈴ちゃんと〜 ( ¨̮ )







公式サイトに私たちが撮影している
メイキング動画もありますので気になる方は
チェックしてみてください〜 。








ちなみにファンクラブ会員限定の
オンライン販売になっています 。

ご予約は開始されているみたいなので、
是非 〜( ¨̮ )














ではでは今日はこのへんで !



最後まで読んで下さり、有難うございました !






おしまい 🌱
