森田 ひかる





みなさん、こんにちは
福岡県出身、18歳の森田ひかるです 🌱








12月に突入して、最近更に寒いですね 。

風邪ひいてないですか?大丈夫ですか? ( ¨̮ )



私はこの前、もっこもこのニットに
ふっわふわのコートを着て出かけたのですが、
汗だくでした 。 (笑)

その日は、そこまで寒くなかったみたいで …

難しいですね〜 。







私は冬の食べ物がとても好きなので、
食欲が止まりません 。

食欲の秋より食欲の冬です ( ¨̮ )




しゃぶしゃぶ食べたくて仕方がない!

おでんもシチューもお鍋も食べたい〜 。




お鍋はやっぱり水炊き派ですが、
カニ鍋にとっても興味があります 🦀

北海道でいつか食べたいな〜 。



皆さんはお鍋食べてますか??





トークアプリのレターでお鍋の写真を
送ってきて下さっている方がいたのですが、
私は寝る前に読んでいるので、飯テロ状態に
いつもなっています 。 (笑)


他にもペットや風景の写真も付けて
送って下さったり、癒されています〜 。
いつも有難うございます (*´`)/








ハシビロコウグッズが少しずつ
増えてきました 。

スリッパもハシビロコウで玄関マットも
ハシビロコウ 。(笑)



リアルなハシビロコウは前に載せたけど
ゆるいのは載せた事なかったので ( ¨̮ )









ここ最近、髪の毛染めた?って
聞かれる事が多いのですが、染めてないです !


私は黒髪が好きなので、染めません〜 。






そういえばこの前、12時間くらい寝て
起きて、鏡みたら寝癖が芸術的すぎて
吹き出しました 。(笑)

洗面所で笑いこけた 。

1人で (笑)




家でよかった〜 、、 (*_*)








そういえば新制服の写真載せた事なかった
ので !


今日の「欅って、書けない?」では
二期生のリアクションチェックです 。


あるメンバーのジェットコースターの
リアクションでお腹抱えて笑いました 。 (笑)

私ももう一度見たいので、見ます 。




ぜひ〜 ( ¨̮ )















あ、そしてそしてこの前のブログで
今年中にブログ目指せ10個以上ー!

って言ってたけどそれから今回で
5個目〜 (*´`)/



思ってたよりかけてる気がします。





せっかく更新して読んで頂いてるから
少しでも長い方がいいかな〜と思って
いつも心がけてますが内容薄くて
ゆるい時もあります 。 (笑)


それはお許しください ( ¨̮ )

















そして最後になりますが、一つだけお知らせ !



「坂道テレビ」の第2弾が12月30日に
放送決定いたしましたー!


また後日詳細は公開されていきますので、
要チェックです ( ¨̮ )


今年の3月に第1弾が放送されましたが、
また2019年最後に放送されるとのことで、
とても嬉しいです 。


ぜひ見て下さい 〜〜










ではでは今日はこのへんで 。





最後まで読んで下さり、有難うございました!

まだ数えてない方は下にスクロールしないで
下さい〜
