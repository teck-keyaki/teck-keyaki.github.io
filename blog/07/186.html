休みの日☆彡

小林 由依




1人で駅で、
どこの出口から出たらいいのかわからなくて
うろうろしてたらオダナナとみーちゃんが丁度来て
助けてくれました。


小林由依です♪




こんばんは〜\( ˙▿︎˙ )/
今日も1日お疲れ様です♡




今日はレッスンでした!
先生から芸能界の色んなことお聞きできるので
すっごく勉強になります

もっともっと頑張らなくちゃ!




そしてなんか今日は特に書くことがないので…
もっと細かな自己紹介しようかな♪♪

でも皆さんは何が知りたいんだろう?


ん〜
今日はお休みの日何をしてるか話しますね!



お休みの日は、大抵10時〜12時の間に起きます
本当は8時くらいに起きるんですけど、
どうしても二度寝しちゃうんです(　˙-˙　)


起きたら、朝ご飯とお昼ご飯を一緒に食べて、
テレビを見ながらゴロゴロします(　˙-˙　)


そのあとは、連続ドラマが大好きなので、
録画してあるドラマを見たり…
ちょろっとギターを弾いたり…
スマホゲームをしたり…

ちなみに、学園もののドラマが一番好きです!
ギターは、井上苑子さんや大原櫻子さんが好きなので、よくその曲を弾きながら大声で歌ってます!
スマホゲームは、野菜を育てたり、ニワトリを飼ったり、船にそれを積むお仕事をするような牧場ゲームが大好きです♪ 楽しいです♪


(　˙-˙　)。


そうこうしているうちに夜になっているので、
夜ご飯を食べて、
バライティー番組やドラマを見ます(　˙-˙　)


そして、お風呂に入ったあと、
音楽を聴きながら夢の世界にGOです(　˙-˙　)



お分かりですか?
地味〜な生活してます笑
外にはほとんど出ないです

でも私は外でパーっと遊ぶより
家でゆっくりまったりしている方が好きなんです♡
もちろん、外で遊ぶことも嫌いじゃないですよ!


そんなお休みの日はいつも
(　˙-˙　)こんな顔して1日過ごしてます〜


ぽんっ



こんな感じ〜(　˙-˙　)


でも今は、ありがたいことにお休みの日が少なくて、
いつもみんなと笑って過ごしてます!

どっちの生活もすっごく楽しいです♪♪


私のこと少し知っていただけましたか?




では、これから私は久しぶりに
牧場ゲームやって来ます!

今日中にレベル上がるかな〜\( ˙▿︎˙ )/




おやすみなさい♡
see you again ⊿⊿
