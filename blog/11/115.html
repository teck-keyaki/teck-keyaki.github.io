感謝の気持ちでいっぱいです

菅井 友香

皆さんこんばんは☆

今日お陰様で、無事にお見立て会を終える事が出来ました!!!

今までの人生で、一番濃い2日間でした。

私達の姿を見る為に足を運んで下さった方々、本当にどうもありがとうございましたT^T
また、応援の気持ちを送って下さった方々もどうもありがとうございました( ;  ; )

初めて欅坂46として1つの大きな事に挑戦させて頂けたと感じる事が出来ました。
最終部でメンバーの思い思いの自己紹介を見ていたら、うるっとしちゃいました( ;  ; )ほとんどが私より年下なのに、凄いなーと思いました(T_T)
そして、このグループの一員として活動できる幸せを噛み締めました。

私の自己紹介なのですが、沢山の方々にアドバイスを頂き、試行錯誤して頑張って練習してきました!!4回ともちょっと内容変えたんですよ〜〜^_^
いざステージに立つと、噛んじゃったりセリフが飛んじゃったりしました>_<
本当に、まだまだ未熟者です。
そんな中でも会場から温かい声援を頂き、凄く励まされました( ;  ; )

そしてそして、
握手に来てくださった方々、本当に本当にありがとうございました♡!!
皆さんとても優しくして下さって、感激してしまい、実は幕にはけてから、ステージでは泣くまいと堪えていた涙が溢れ出てきちゃいましたT^T

私は実は、普段涙を人に見せないように我慢するタイプなんです。
でも、来て下さった方々にかけて頂いた暖かい言葉の数々、私達が集合する前から準備をし、夜遅くまで私達をサポートして下さっているスタッフの方々の姿に心を打たれてしまい、暫く涙が止まらなくなってしまいました( ;  ; )
こんな感情は初めてで、自分でも戸惑ってしまう程でした。

初めてステージの裏側を知り、私達のお見立て会を開催させて頂くにあたり、こんなにも沢山の方々が動いて下さっているのだと目の当たりにしました。
私達が自分の事に集中出来るようにと環境を整えて下さった関係者の皆様、本当にありがとうございました。

いつまでも、この感謝の気持ちを忘れずにいます。

そして、尊敬している乃木坂46さんの先輩である伊藤かりんさん、美味しくて可愛い差し入れをありがとうございましたT^T♡
涙の塩味と共に美味しく頂きました!!

今回思いがけず頂いた素敵なお花は大切にお持ち帰りさせて頂きました꒰ ´͈ω`͈꒱

愛猫のトムも興味津々です!!
このお花が、明日も明後日もお見立て会の興奮を思い出させてくれそうです꒰ ´͈ω`͈꒱
本当にありがとうございます!!!

最後に、本番前のお写真パシャリ
ななこが100万馬力ポーズしてくれました♡

りなのトランペットは何回聴いても爆笑でした笑!


お伝えしたい事が多すぎて、長くなってしまいました汗
最後まで読んで下さりありがとうございました!

こんな私ですが、明日からもレッスンに励んで一生懸命頑張っていきますので、応援して頂けましたらすごくすごく嬉しいです꒰ ´͈ω`͈꒱

私はまだ興奮して眠れなそう!
皆さんはゆっくり眠れますように♪
おやすみなさい☆
